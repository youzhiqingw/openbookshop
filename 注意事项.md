## 📝 开发规范

### 后端规范

* 遵循 PEP 8 代码风格
* 使用类型提示 (Type Hints)
* 视图使用 CBV 或 ViewSets
* 序列化器使用 ModelSerializer
* 权限检查使用自定义 Permission 类

### 前端规范

* 使用 Composition API + `<script setup>` 语法
* 组件名使用大驼峰 (PascalCase)
* API 请求统一封装，使用拦截器处理 Token
* 路由使用懒加载
* 状态管理使用 Pinia
