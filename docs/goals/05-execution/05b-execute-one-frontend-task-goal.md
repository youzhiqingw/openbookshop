# 05b Execute One Frontend Task Goal

使用前把 `<TASK-ID>` 替换为 docs/TASKS.md 中的具体任务 ID。

```text
/goal implement only frontend task <TASK-ID> until:

- read AGENTS.md, docs/TASKS.md, and the matching docs/specs file
- read only affected web/src files before editing
- change only files owned by <TASK-ID>
- use Vue3 TypeScript, Pinia, Element Plus/FastCrud patterns
- keep package manager as pnpm
- run focused validation: pnpm run build if feasible
- report modified files and validation result

Do not start another task.
Stop after this task is complete or blocked.
```
