# 03 SPEC 微型 Goal

这里的 goal 用来解决 03 SPEC 阶段上下文过长的问题。每次只运行一个模块 goal，输出一个 `docs/specs/<module>.md`。

推荐顺序：

1. `03a-foundation-spec-goal.md`
2. `03b-merchant-spec-goal.md`
3. `03c-catalog-spec-goal.md`
4. `03d-order-spec-goal.md`
5. `03e-payment-review-spec-goal.md`
6. `03f-frontend-spec-goal.md`

原则：

- 只读必要文件。
- 只产出一个 SPEC。
- 不把长文档粘入工具命令。
- 不实现代码。
