# 04a Task One SPEC Goal

使用前把 `<module>` 替换为 `foundation`、`merchant`、`catalog`、`order`、`payment-review` 或 `frontend`。

```text
/goal break only docs/specs/<module>.md into executable tasks until:

- read AGENTS.md, docs/specs/<module>.md, docs/TASKS.md
- each task has ID, goal, files owned, dependencies, steps, validation, done criteria, risk, rollback
- each task owns at most 3 files unless justified
- tasks are ordered by dependency
- update only the <module> section in docs/TASKS.md
- identify the exact next task ID

Do not write implementation code.
Stop after tasks for this module are complete or after 6 turns.
```
