# 前端設計插件

生成獨特的、生產級別的前端界面，避免通用的 AI 美學。

## 功能

Claude 會自動將此技能用於前端工作。創建具有以下特點的生產就緒代碼：

- 大膽的美學選擇
- 獨特的排版和調色板
- 高影響力的動畫和視覺細節
- 感知上下文的實現

## 用法

```
"為音樂流媒體應用創建儀表板"
"為 AI 安全初創公司構建登陸頁面"
"設計具有深色模式的設置面板"
```

Claude 將選擇清晰的美學方向，並實現對細節一絲不苟的生產代碼。
